## 🔍 常见问题排查

### 问题 1：编辑器无法切换

**症状**：点击切换按钮无反应

**排查步骤**：
1. 打开浏览器控制台（F12）
2. 查看是否有 JavaScript 错误
3. 检查 `@uiw/react-md-editor` 是否正确安装

**解决方案**：
```bash
cd company_cms_project/frontend
npm install @uiw/react-md-editor react-markdown
```

---

### 问题 2：分类/标签选择器为空

**症状**：下拉框中没有选项

**排查步骤**：
1. 检查后台是否创建了分类/标签
2. 打开 Network 面板查看 `/api/v1/posts/categories-tags` 请求
3. 验证返回的数据格式

**解决方案**：
- 先在后台创建分类和标签
- 或直接在文章编辑时输入新标签

---

### 问题 3：Markdown 内容不渲染

**症状**：文章内容显示原始 Markdown 语法

**排查步骤**：
1. 检查文章的 `content_format` 字段值
2. 查看 PostDetailPage 组件的渲染逻辑
3. 验证 `react-markdown` 是否正确导入

**解决方案**：
```tsx
// 确保渲染逻辑包含 content_format 判断
if (post.content_format === 'markdown') {
  return <ReactMarkdown>{post.content}</ReactMarkdown>;
} else {
  return <div dangerouslySetInnerHTML={{ __html: post.content }} />;
}
```

---

### 问题 4：API 请求 401 错误

**症状**：前台访问文章列表时报错

**原因**：使用了需要认证的 API

**解决方案**：
- 前台使用 `/api/v1/posts/public` 接口
- 后台使用 `/api/v1/posts` 接口
