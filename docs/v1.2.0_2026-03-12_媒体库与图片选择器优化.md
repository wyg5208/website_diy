## 一、问题修复

### 1.1 媒体库图片预览失败

**问题描述**：  
在媒体库管理页面（`/admin/media`），上传的图片无法显示预览，点击眼睛图标也无法查看大图。后端日志显示 `/media/2026/03/xxx.jpg` 返回 404。

**问题原因**：  
Flask 后端未配置媒体文件的静态路由，导致前端通过 `http://127.0.0.1:5000/media/...` 访问图片时返回 404。

**修复方案**：  
在 `backend/app/__init__.py` 中添加媒体文件静态路由：

```python
@app.route('/media/<path:filename>')
def serve_media(filename):
    """提供媒体文件访问"""
    upload_folder = os.path.abspath(app.config['UPLOAD_FOLDER'])
    file_path = os.path.normpath(os.path.join(upload_folder, filename))
    
    # 安全检查：确保文件路径在上传目录内
    if not file_path.startswith(upload_folder):
        return {'code': 403, 'message': '非法访问'}, 403
    
    if not os.path.exists(file_path):
        app.logger.warning(f'文件不存在: {file_path}')
        return {'code': 404, 'message': '文件不存在'}, 404
        
    return send_from_directory(upload_folder, filename)
```

**涉及文件**：
- `backend/app/__init__.py`

### 1.2 媒体库视频无法预览播放

**问题描述**：  
上传的视频文件（如 MP4）在媒体库中无法显示缩略图，点击预览也无法播放。

**问题原因**：  
`MediaManager.tsx` 组件对所有媒体类型统一使用 `<img>` 标签渲染，无法正确处理视频文件。

**修复方案**：  
1. 添加 `previewType` 状态，区分图片和视频
2. 根据 `mime_type` 判断文件类型，视频使用 `<video>` 标签
3. 预览弹窗中视频显示播放器控件

**核心代码变更**：

```tsx
// 列表卡片封面
{item.mime_type?.startsWith('video/') ? (
  <video
    src={`http://127.0.0.1:5000${item.file_url}`}
    style={{ maxWidth: '100%', maxHeight: '100%', objectFit: 'contain' }}
    preload="metadata"
  />
) : (
  <img alt={item.filename} src={`http://127.0.0.1:5000${item.file_url}`} />
)}

// 预览弹窗
{previewType === 'video' ? (
  <video src={previewUrl} controls autoPlay style={{ width: '100%' }} />
) : (
  <img alt="preview" src={previewUrl} style={{ width: '100%' }} />
)}
```

**涉及文件**：
- `frontend/src/pages/MediaManager.tsx`

## 二、新功能开发

### 2.1 页面编辑器图片选择器组件

**需求描述**：  
页面编辑器中的图片组件，需要支持三种方式选择图片：
1. 从媒体库选择已上传的图片
2. 从本地上传新图片（自动保存到媒体库）
3. 输入网络图片 URL

**实现方案**：  
创建独立的 `ImagePicker` 组件，集成到页面编辑器中。

**组件功能**：
| 功能 | 说明 |
|------|------|
| 媒体库选择 | 显示已上传图片网格列表，点击选中 |
| 本地上传 | 拖拽或点击上传，自动保存到媒体库 |
| 网络图片 | 输入 URL，实时预览 |
| 已选预览 | 显示当前选中图片缩略图和文件名 |

**组件接口**：
```tsx
interface ImagePickerProps {
  value?: string;           // 当前图片 URL
  onChange?: (url: string) => void;  // URL 变更回调
  placeholder?: string;     // 占位文字
}
```

**涉及文件**：
- `frontend/src/components/ImagePicker.tsx` (新增)
- `frontend/src/pages/PageEditor.tsx` (修改)
