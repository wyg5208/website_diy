## 🐛 Bug 修复

### 1. 文章保存不成功问题
**问题描述**: 新建文章保存后不显示

**原因分析**:
- 默认状态为 `draft`（草稿），不会显示在前台
- 保存成功后页面未刷新

**解决方案**:
- 修改默认值为 `published`
- 添加"立即发布"和"存为草稿"两个按钮
- 优化错误提示，显示详细 API 错误信息

---

### 2. 空值处理异常
**问题描述**: 某些字段为空时报 AttributeError

**解决方案**:
```python
# 修改前
title = data.get('title', '').strip()

# 修改后
title = data.get('title', '').strip() if data.get('title') else ''
```

---

### 3. TypeScript 类型错误
**问题描述**: CardProps 类型导入错误

**解决方案**: 移除未使用的类型导入

## 💡 使用指南

### 1. 新建并发布文章
```
访问：http://localhost:5173/admin/posts/new
→ 填写标题和内容
→ 选择分类和标签
→ 点击"立即发布"
→ 返回列表查看新文章
```

### 2. 在页面中展示文章
```
访问页面编辑器
→ 点击左侧"📄 文章列表"
→ 配置显示模式、筛选条件
→ 保存并发布
→ 前台页面自动显示
```

### 3. 配置文章列表组件
```yaml
显示模式：列表 / 网格
显示标题：是/否
自定义标题：例如"公司新闻"
每页显示：1-50
显示分页：是/否
按分类筛选：输入分类 ID
按标签筛选：输入标签名称或 ID
```
