# v1.5.1 容器子组件新增文章列表选项

**发布日期**: 2026-03-17  
**版本号**: v1.5.1  
**更新类型**: 功能增强

---

## 更新概述

本次更新为页面编辑器容器组件新增"文章列表"子组件选项，并优化了文章列表组件的分类/标签选择器，从手动输入改为下拉选择，提升用户体验。

---

## 功能变更

### 1. 容器子组件新增文章列表选项 📝

**变更文件**: `frontend/src/pages/PageEditor.tsx`

**变更内容**:
- 在顶层容器的子组件选项中添加"文章列表"
- 在嵌套容器的子组件选项中添加"文章列表"
- 文章列表组件支持在容器内灵活布局

**代码变更**:
```tsx
// 顶层容器子组件选项
const baseOptions = [
  { value: 'text', label: '文本块' },
  { value: 'image', label: '图片' },
  { value: 'button', label: '按钮' },
  { value: 'cards', label: '卡片列表' },
  { value: 'articles', label: '文章列表' },  // 新增
  { value: 'divider', label: '分割线' },
];

// 嵌套容器子组件选项
options={[
  { value: 'text', label: '文本块' },
  { value: 'image', label: '图片' },
  { value: 'button', label: '按钮' },
  { value: 'cards', label: '卡片列表' },
  { value: 'articles', label: '文章列表' },  // 新增
  { value: 'divider', label: '分割线' },
]}
```

### 2. 文章列表分类/标签选择器优化 🏷️

**变更文件**: `frontend/src/pages/PageEditor.tsx`

**变更内容**:
- 新增分类和标签数据加载逻辑
- 分类筛选从输入框改为下拉选择器
- 标签筛选从输入框改为下拉选择器
- 支持搜索过滤、清空选择

**新增状态**:
```tsx
const [categories, setCategories] = useState<any[]>([]); // 分类列表
const [tags, setTags] = useState<any[]>([]); // 标签列表
```

**新增 API 调用**:
```tsx
// 加载分类和标签列表
useEffect(() => {
  const loadCategoriesAndTags = async () => {
    try {
      const res = await getCategoriesAndTags();
      setCategories(res.data.categories || []);
      setTags(res.data.tags || []);
    } catch (error) {
      console.error('Failed to load categories and tags:', error);
    }
  };
  loadCategoriesAndTags();
}, []);
```

**UI 变更**:
```tsx
// 分类选择器
<Select 
  value={props.categoryId || undefined} 
  onChange={(v) => handlePropsChange('categoryId', v)} 
  placeholder="选择文章分类"
  allowClear
  showSearch
  filterOption={(input, option) => 
    (option?.label ?? '').toLowerCase().includes(input.toLowerCase())
  }
  options={[
    { value: undefined, label: '全部分类' },
    ...categories.map(cat => ({ value: cat.id, label: cat.name }))
  ]}
/>

// 标签选择器
<Select 
  value={props.tagId || undefined} 
  onChange={(v) => handlePropsChange('tagId', v)} 
  placeholder="选择文章标签"
  allowClear
  showSearch
  filterOption={(input, option) => 
    (option?.label ?? '').toLowerCase().includes(input.toLowerCase())
  }
  options={[
    { value: undefined, label: '全部标签' },
    ...tags.map(tag => ({ value: tag.id, label: tag.name }))
  ]}
/>
```

## 用户体验提升

| 优化项 | 变更前 | 变更后 |
|--------|--------|--------|
| 分类筛选 | 手动输入分类 ID | 下拉选择分类名称 |
| 标签筛选 | 手动输入标签名称/ID | 下拉选择标签名称 |
| 搜索功能 | 无 | 支持关键词搜索过滤 |
| 清空选择 | 删除输入内容 | 点击清空按钮 |
