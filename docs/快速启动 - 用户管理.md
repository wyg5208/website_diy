# 用户管理功能 - 快速启动指南

## 🚀 立即使用

### 步骤 1: 启动后端服务

打开一个新的终端窗口：

```bash
cd backend
python run.py
```

看到以下输出表示后端启动成功：
```
* Running on http://127.0.0.1:5000
```

### 步骤 2: 启动前端服务

打开另一个新的终端窗口：

```bash
cd frontend
npm run dev
```

看到以下输出表示前端启动成功：
```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

### 步骤 3: 访问用户管理页面

1. 打开浏览器访问：http://localhost:5173/login
2. 使用默认管理员账户登录：
   - **用户名**: admin
   - **密码**: admin123
3. 登录后，点击左侧菜单的"用户管理"
4. 或直接访问：http://localhost:5173/admin/users

## ✅ 功能验证

进入用户管理页面后，您应该能看到：

1. **用户列表**：显示当前所有用户（至少有一个 admin 用户）
2. **添加用户按钮**：右上角有蓝色的"添加用户"按钮
3. **操作选项**：每个用户后面有"编辑"、"重置密码"、"删除"按钮

### 测试添加用户

1. 点击右上角的"添加用户"按钮
2. 填写表单：
   - 用户名：testuser
   - 密码：test1234
   - 邮箱：test@example.com
   - 显示名称：测试用户
   - 状态：保持开启（正常）
3. 点击"确定"
4. 如果成功，列表中会出现新用户

### 测试其他功能

- **编辑用户**：点击任意用户的"编辑"按钮，修改信息后保存
- **重置密码**：点击"重置密码"，在抽屉中输入新密码
- **删除用户**：点击"删除"，确认后删除该用户

## 🔧 故障排查

### 问题 1: 后端启动失败

**可能原因**：缺少 Python 依赖

**解决方案**：
```bash
cd backend
pip install -r requirements.txt
```

### 问题 2: 前端启动失败

**可能原因**：缺少 Node.js 依赖

**解决方案**：
```bash
cd frontend
npm install
```

### 问题 3: 登录时提示"用户名或密码错误"

**可能原因**：数据库中没有 admin 用户

**解决方案**：运行检查脚本或手动创建用户
```bash
cd backend
python check_users.py
```

如果没有用户，可以运行：
```bash
python test_login.py
```

或者使用我们提供的 API 测试脚本自动创建：
```bash
cd d:\python_projects\company_cms
python test_users_api.py
```

### 问题 4: 页面空白或报错

**可能原因**：路由配置未生效

**解决方案**：
1. 清除浏览器缓存
2. 硬刷新页面（Ctrl + F5）
3. 检查浏览器控制台是否有错误信息

## 📝 API 测试
